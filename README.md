# PL IMMOBILIEN – Neue Website

**Immobilien mit Leidenschaft**

Moderne, elegante und hochperformante Next.js-Website für PL IMMOBILIEN (Patrizia Steyskal) in Dietikon/Zürich.

## ✨ Highlights

- **Kreatives Luxus-Design** mit Playfair Display Serif für Überschriften und warmen Gold-Akzenten
- **Interaktive Objektgalerie** mit Live-Filtern, Suche und Detail-Modals
- **Framer Motion** Animationen für ein Premium-Erlebnis
- **Vollständig optimiert** nach Vercel React Best Practices
- **Netlify Forms** für den Kontakt
- **Responsives Design** mit eleganter Mobile Navigation

## 🚀 Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS + Custom Brand Colors
- Framer Motion
- Lucide Icons
- React Hook Form + Sonner Toasts

## 📦 Lokale Entwicklung

```bash
npm install
npm run dev
```

## 🌐 Deployment

Bereit für Netlify oder Vercel.
